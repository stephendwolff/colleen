Data

‘We Need Us’ uses classification and user data from Zooniverse.org . The data is accessed in real-time via an a API (application programming interface). Each time a user logins and classifies a piece of data on any project in the Zooniverse, some metadata about that action is stored in a database and is published as a live feed.

‘We Need Us’ takes the Zooniverse metadata and stores it in a useful format in a new database. The metadata is processed to create more metadata to provide dynamic input into the sound and animation engines we use.

You can see the current live data in our chart tool.
